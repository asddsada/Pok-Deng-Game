# Pok-Deng-Game
Skooldio Backend Homework

## Requirements
- nodejs: v14.15.1
- mathjs: ^26.6.3
- jest: ^8.0.1

## Initialize
1. clone git
2. start pipeline
    ```
    npm install
    ```

## Usage
```
    npm start

    # example
    # Please put your bet
    # $ 5
    # You got Clubs-King, Hearts-9
    # The dealer got Spades-4, Spades-4
    # You won!!!, received 5 chips
    # Wanna play more (Yes/No)?
    # $ No
    # You got total 5 chips

```
